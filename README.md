# OneEateries
  Created simple get and post request using spring boot
  Get Request is "/oneeateries/rest/restaurant/menu/"
  Post Request is "/oneeateries/rest/restaurant/menu"
  A simple post request body:-
    {
	    "restaurantName": "A",
	    "menuName": "B",
	    "menuItem": [
		    {
			    "itemName": "C",
			    "description": "D",
			    "price": "100"
		    },
		    {
			    "itemName": "E",
			    "description": "F",
			    "price": "200"
		    }
	    ]
    }
